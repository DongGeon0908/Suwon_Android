##### 3단원 : 기본 위젯과 드로어블 사용하기

- 기본위젯
    + 버튼
    + 텍스트뷰
    + 에디트텍스트
    + 이미지뷰

- 사용자에게 간단한 정보를 전달
    + 토스트
    + 대화상자
    + 프로그래스바

- 텍스트뷰 
    + 화면에 글자를 보여주는 역할
    + `text` 속성
        * 텍스트뷰의 문자열 설정
        * `text`속성은 반드시 지정
        * 텍스트뷰에 문자열이 없으면 텍스트뷰가 차지하는 영역 알 수 없음
        * `xml layout` 또는 `strings.xml`을 통해 설정
        * 다국어 지원에 효율적
    + `textColor` 속성
        * 텍스트뷰에서 표시하는 문자열의 색상
        * `#AARRGGBB` 포맷 사용
    + `textSize` 속성   
        * 텍스트뷰에서 표시하는 문자열의 크기 설정
        * `sp, dp, px`
    + `textStyle` 속성
        * 텍스트뷰에서 표시하는 문자열의 스타일 속성 설정
        * `normal, bold, ital-ic`
        * `|`기호를 통해 다양한 값 지정 가능
    + `typeFace` 속성
        * 텍스트뷰에서 표시하는 문자열의 폰트 설정
        * `normal, sans, serif, monospace`
    + `maxLines` 속성
        * 텍스트뷰에서 표시하는 문자열의 최대 줄 수를 설정
        * 한 줄로만 표시하고 싶을 때는 값을 "1"로 설정

- 버튼
    + 사용자가 클릭하면 클릭에 대한 반응을 하는 위젯
    + 텍스트뷰를 상속
    + `OnClickListener`를 통해 위젯에 발생한 이벤트 처리

- 에디트텍스트
    + 사용자에게 값을 입력받을 때 사용
    + 입력하는 문자의 유형을 지정할 수 있는 `inputType` 속성 제공
    + `hint` 속성을 통해 간단한 안내글 표시

- 이미지뷰와 이미지 버튼
    + 이미지를 화면에 표시할 때 사용하는 간단한 위젯
    + `android:src`와 `app:srcCompat`
        * 원본 이미지를 설정
    + `maxWidth`와 `maxHeight`
        * 이미지가 표시되는 최대 폭, 높이를 설정
    + `tint` 
        * 이미지뷰에 보이는 이미지의 색상을 설정
        * "#AARRGGBB" 포맷 사용
    + `scaleType`
        * 이미지뷰의 크기에 맞게 원본 이미지의 크기를 자동으로 늘리거나 줄여서 보여줄 때 사용
        * `fitXY, centerCrop, centerInside`등의 이미지 변환 알고리즘이 적용된 미리 정의된 값을 사용

- 텍스트뷰와 에디트텍스트의 다른 기능
    + 뷰를 상속 받음
    + 커서 관련 속성
        * 문자나 숫자를 입력하는 역할을 하며 커서가 깜박이는 동작
        * `selectAllOnFocus` 속성을 `true`로 설정하면 포커스를 받을 때 문자열 전체가 선택
    + 자동 링크 관련 속성  
        * `autoLink` 속성을 `true`로 설정하면 문서에 포함된 웹페이지 주소나 이메일 주소를 링크 색상으로 표시
        * 링크를 누르면 웹페이지에 바로 접속하거나 메일 편집기를 띄워주는 기능 제공
    + 줄 간격 조정 관련 속성
        * 텍스트뷰의 줄 간격을 조절
        * `lineSpacingMultiplier`와 `lineSpacingExtra` 
        * 기본 줄 간격의 배수로 설정 
        * 여유값을 설정할 떄 사용
    + 대소문자 표시 관련 속성
        * `capitalize` 속성을 이용하면 글자, 단어, 문장 단위로 대소문자 조절
    + 줄임 표시 관련 속성
        * `ellipsize` 속성을 사용하면 입력한 내용의 생략 부분을 설정
    + 힌트 표시 관련 속성
        * `hint`는 에디트텍스트에 어떤 내용을 입력하라고 안내문으로 알려주고 싶을 때 사용
    + 편집 기능 관련 속성
        * 에디트텍스트에 입력되어 있는 문자열을 편집하지 못하게 하고 싶다면 `editable` 속성 값을 `false`로 설정
    + 문자열 변경 처리 관련 속성
        * 에디트텍스트에 입력된 문자를 확인하거나 입력된 문자가 필요한 포맷과 맞는지 확인할 때 getText() 메서드 사용

- 뷰의 배경 이미지
    + `background` 속성은 배경색을 설정하거나 이미지 파일을 설정할 때 사용

- 드로어블
    + 뷰에 설정할 수 있는 객체이며 그 위에 그래픽을 그릴 수 있음
    + `BitmapDraWable`
        * 이미지 파일을 보여줄 때 사용
    + `StateListDrawable`
        * 상태별로 다른 그래픽을 참조
    + `TransitionDrawable`
        * 두 개의 드로어블이 서로 바뀌도록 만듬
    + `ShapeDrawable`
        * 색상과 그라데이션을 포함해 도형 모양을 정의
    + `InsertDrawable`
        * 지정한 거리만큼 안쪽으로 들어오도록 만듬
    + `Clip-Drawable`
        * 다른 드로어블을 클리핑
    + `ScaleDrawalbe`
        * 다른 드로어블의 크기를 바꿈

- 이벤트 처리 방식
    + 터치 이벤트를 쉽게 처리할 수 있도록 클릭 이벤트(Click Event) 제공
    + 위임 모델 `Delegation Model`
        * 화면에서 발생하는 이벤트를 버튼과 같은 위젯 객체에 전달한 후 그 이후의 처리 과정을 버튼에 위임
    + 터치가 되었을때, 터치를 하지 않을 때 등 모든 동작에 대한 이벤트를 제공

- 키 이벤트
    + `onKeyDown()` 메서드를 재정의
    + KeyCode는 어떤 키가 사용되는지 구별할 때 사용
    + KeyEvent는 키 입력 이벤트에 대한 정보를 알고 싶을 때 사용

- 수명 주기`Life Cycle` or 생명 주기 메서드
    + 화면의 상태에 따라 시스템이 자동으로 호출해주는 메서드

- 토스트
    + 간단한 메시지를 잠깐 보여주었다가 없어지는 뷰
    + 앱 위에 떠 있는 뷰
    + `Context`
        * `Context`클래스를 상속한 액티비티를 사용
    + `setGravity()`
        * 토스트 뷰가 보이는 위치를 지정하는 데 사용

- 스낵바
    + 머티리얼 라이브러리(Material Library)를 추가해야 사용 가능
    + Toast처럼 간단하게 메시지를 사용자에게 제공

- 알림 대화상자
    + 사용자에게 확인을 받거나 선택하게 할 때 사용
    + 일방적으로 메시지를 전달하는 역할

- 프로그레스바
    + 작업의 진행 정도를 표시하거나 작업이 진행 중임을 사용자에게 알려줌
